# Knowledge Base Content

This folder holds the documents that power the assistant's `search_knowledge_base` tool.

These are **sample procedures** meant to be replaced with your own. Drop in Markdown, PDF, or text files describing the procedures, policies, and troubleshooting guides relevant to your industry, then sync them to your Bedrock Knowledge Base (see `scripts/create-knowledge-base.sh`).

The assistant retrieves the most relevant chunks at query time and answers conversationally — so write the content the way you'd want it explained to a frontline worker.

## Tips
- Keep each document focused on a single topic
- Use clear headings and step-by-step structure
- Include specific values (contact numbers, time limits, thresholds) where relevant
- No special formatting required — plain prose works well for RAG
